## ­Ъџђ What It Does

- Displays a collection of images in a horizontally scrollable container.
- Incorporates smooth scrolling and transition effects for a fluid user experience.
- Adapts seamlessly to various screen sizes and input methods.
- Offers an enhanced visual design with refined gradients and spacing.

## ­ЪДа How It Works

- Utilizes `display: flex` with `overflow-x: scroll` for horizontal layout.
- JavaScript listens for `wheel` events to control horizontal scrolling.
- Navigation buttons allow manual scroll control.
- CSS `scroll-behavior: smooth` enables smooth scrolling transitions.
- Responsive design achieved through media queries and flexible sizing.

## ­ЪЊї рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў

- Scroll snapping рЃ»рЃћрЃа рЃљрЃа рЃљрЃарЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў
- рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃА рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ рЃљрЃарЃљрЃљ рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў
- рЃљрЃарЃљ рЃљрЃЦрЃЋрЃА dark mode (рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃЊрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃћрЃџрЃўрЃљ)

------

## ­ЪДф рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃљ

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃдрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃЌрЃљрЃљ Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ, fork-рЃўрЃЌ, рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ, рЃњрЃљрЃЊрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ.  
рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ рЃљрЃЋрЃбрЃЮрЃарЃў, рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃљ. Рюї№ИЈ

---

## ­ЪїЪ рЃЏрЃ«рЃљрЃарЃў рЃЊрЃљрЃБрЃГрЃўрЃарЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃА

рЃЌрЃБ рЃЏрЃЮрЃњрЃћрЃгрЃЮрЃюрЃљрЃЌ рЃљрЃю рЃњрЃљрЃЏрЃЮрЃњрЃљрЃЊрЃњрЃљрЃЌ, рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ РГљ GitHub-рЃќрЃћ.  
рЃњрЃљрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЌ, рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ, рЃЊрЃљрЃњрЃЋрЃљрЃЌрЃћрЃњрЃћрЃЌ Рђћ рЃћрЃарЃЌрЃљрЃЊ рЃЋрЃЦрЃЏрЃюрЃўрЃЌ рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃў рЃАрЃЎрЃарЃЮрЃџрЃўрЃА рЃЏрЃљрЃњрЃўрЃљрЃА!

---

## РюЇ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў

рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЏрЃЦрЃюрЃћрЃџрЃў:  
**рЃЊрЃљрЃЋрЃўрЃЌ рЃ»рЃљрЃюрЃћрЃќрЃљрЃерЃЋрЃўрЃџрЃў**  
_aka MasterJaneza РђЊ рЃЎрЃарЃћрЃљрЃбрЃўрЃБрЃџрЃў рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃў рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃћрЃарЃў_

## Рџа№ИЈ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ

­ЪЊ▒ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃдрЃўрЃљ рЃарЃћрЃЪрЃўрЃЏрЃў.  
рЃќрЃЮрЃњрЃўрЃћрЃарЃЌрЃў рЃњрЃарЃљрЃЊрЃўрЃћрЃюрЃбрЃў рЃљрЃю рЃЕрЃарЃЊрЃўрЃџрЃў рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃЏрЃБрЃЦ рЃарЃћрЃЪрЃўрЃЏрЃерЃў.
